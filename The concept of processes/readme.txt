# Описание программы

Данный проект состоит из двух программ: `parent` и `child`. 

## Функциональность

### Программа `parent`

- Создает дочерние процессы, используя системные вызовы `fork()` и `execve()`.
- Формирует сокращенную среду для дочерних процессов на основе переменных окружения, указанных в файле `env`.
- Выводит переменные окружения, установленные оболочкой, отсортированные в соответствии с `LC_COLLATE=C`.
- Обрабатывает нажатия клавиш:
  - Нажатие `+` создает новый дочерний процесс и запускает программу `child`.
  - Нажатие `*` создает еще одного дочернего процесса, который получает переменные окружения из текущего окружения.
  - Нажатие `q` завершает выполнение родительского процесса.

### Программа `child`

- Выводит свое имя, PID (идентификатор процесса) и PPID (идентификатор родительского процесса).
- Читает переменные окружения из файла `env` и выводит их значения.
- Если запущен с аргументами, выводит все переменные окружения, доступные в текущем процессе.


###Инструкция по настройке программы

1. Перейдите в директорию laba02 и соберите проект:
       make
   Чтобы собрать в режиме отладки:
       make MODE=debug
   Чтобы собрать в режиме выпуска:
       make MODE=release
       
2. Настройте переменную CHILD_PATH:
export CHILD_PATH=/ваш/путь/до/исполняемого/файла/child

3. Для запуска выполните следующее и используйте parent:
./parent
